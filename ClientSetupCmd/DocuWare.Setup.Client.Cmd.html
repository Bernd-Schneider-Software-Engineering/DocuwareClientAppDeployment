DocuWare.Setup.Client.Cmd.exe

This utility enables you to install DocuWare Client Applications in an automated or silent way.

Prerequisites

  • You need to have Microsoft .NET 4.7.2 or higher installed on the target environment.
  • All other DocuWare prerequisites (like C++ runtimes) are automatically installed by DocuWare.Setup.Client.Cmd.exe
This utility can be used with DocuWare 6.8 or above. A new version will be provided with each release.

Further information

For more information run one of the following commands: 
  • DocuWare.Setup.Client.Cmd.exe help
  • DocuWare.Setup.Client.Cmd.exe help install
  • DocuWare.Setup.Client.Cmd.exe help uninstall
  • DocuWare.Setup.Client.Cmd.exe help upgrade
Or visit https://go.docuware.com/ClientSetupCmd.

Samples

Some common sample commands: 
  • DocuWare.Setup.Client.Cmd.exe install --modules="AdminTool Scanner Printer Import"  --server=http://servername --orgId="00000000-0000-0000-0000-000000000000"
  • DocuWare.Setup.Client.Cmd.exe upgrade --all
  • DocuWare.Setup.Client.Cmd.exe uninstall --all
Deprecated command: 
  • DocuWare.Setup.Client.Cmd.exe install --modules="AdminTool Scanner Printer Import"  --server=http://servername --orgId=1 (The Integer orgId is deprecated please use GUID as above.)
